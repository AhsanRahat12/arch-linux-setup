# Creating an Install Medium

## Overview
This process involves downloading the official Arch Linux ISO and flashing it to a USB drive for installation.

## Steps

### 1. Download the ISO
- Downloaded the ISO from the official Arch Linux website
- Used the country-specific mirror for faster download speeds
- Verified the ISO integrity (recommended for security)

### 2. Flash to USB
- Used **Balena Etcher** to flash the ISO to an empty USB stick
- Alternative tools: `dd` command, Rufus (Windows), or other disk imaging tools
- Ensured USB drive had sufficient space (minimum 2GB recommended)

## Tools Used
- **Balena Etcher**: Cross-platform tool for flashing disk images
- USB drive (8GB+ recommended for convenience)

## Notes
- Always backup any important data on the USB drive before flashing
- The USB drive will be completely wiped during this process
- Verify the download completed successfully before flashing
